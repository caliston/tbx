Drawings

You can output paths to the screen using vector graphics. The tbx::DrawPath encapsulates methods to create and display paths that will be rendered using the RISC OS Draw module.

Using these paths it is possible to draw shapes consisting of bezier curves, control how lines are drawn (including the thickness) and how shapes are filled.

The graphics classes contain methods to draw the outlines (stroke) or fill a draw path positioning it at a given location. They internally set up the transform required by the tbx::DrawPath to position it and to treat the user coordinates used by the path as if they were specified as OS coordinates.
