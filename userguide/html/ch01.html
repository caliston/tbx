Introduction

TBX is a C++ library to help write desktop programs for RISC OS. At its core level it interfaces with the RISC OS toolbox which provides objects to show windows/gadgets and standard dialogue boxes.

As well as this core level it provides other classes to help in the creation of fully featured applications.

It is released under the MIT licence that allows it to be used with open and closed source applications freely.
