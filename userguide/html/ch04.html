The Application class

The tbx has one main class that is used to setup and run a Toolbox application called Application. This class is used to setup the application and initialise the Toolbox, and run the main polling loop that is the heart of any desktop application.

A tbx application has a main subroutine that creates an application object and then runs it.

Example 4.1. Common contents of the main for a tbx routine

int main()
{
    tbx::Application my_app("<MyApp$Dir>");

    // Set up handlers required for the main application
    
    my_app.run();
}
